## 前言

面试是很可怕的，甚至可以让最有经验的专家在压力之下忘记如何回答。一起来了解这些社区收集的面试中常见问题，为面试官提出的任何问题做好准备。通过把经验总结和现实例子结合起来，可以让你从紧张到从容不迫地迎接下一个机会。
